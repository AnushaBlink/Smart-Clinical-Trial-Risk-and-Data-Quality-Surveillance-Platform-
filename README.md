# Smart-Clinical-Trial-Risk-and-Data-Quality-Surveillance-Platform-
Smart Clinical Trial Risk and Data Quality Surveillance Platform for Risk-Based Monitoring and Patient Safety.

A portfolio project demonstrating how modern Clinical Data Management, Pharmacovigilance, and Risk-Based Monitoring can be integrated using Excel, SQL, and Power BI.

Project Overview
This project simulates a clinical trial monitoring environment where patient safety, data quality, protocol compliance, and site performance are analyzed through interactive dashboards and SQL analytics.

Key Features
Clinical trial data management using structured Excel datasets
SQL-based analysis for patient enrollment, adverse events, protocol deviations, delayed visits, and open queries
Dynamic Power BI dashboards for executive insights and operational monitoring
Risk scoring model to identify high-risk patients and study sites
Data quality surveillance for improved clinical decision-making
Risk-Based Monitoring (RBM) approach aligned with modern clinical research practices

Tools & Technologies
Microsoft Excel
SQL
Microsoft Power BI

This project highlights practical skills in clinical data analytics, visualization, healthcare business intelligence, and pharmaceutical data management.
